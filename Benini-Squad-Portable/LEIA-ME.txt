  BENINI SQUAD — Versão portátil para outro desktop
═══════════════════════════════════════════════════════════════

Esta pasta contém o site completo: HTML, CSS, JavaScript, imagens,
vídeo do hero, modelos 3D (GLB), fontes embutidas e todos os efeitos
(GSAP, ScrollTrigger, Lenis, Three.js).

Tamanho aproximado: ~230 MB (modelos 3D + vídeo).

───────────────────────────────────────────────────────────────
COMO ABRIR (recomendado)
───────────────────────────────────────────────────────────────

Os modelos 3D e o vídeo NÃO funcionam ao abrir o HTML direto no
navegador (protocolo file://). Use um servidor local na pasta:

  macOS / Linux:
    • Dê duplo clique em: iniciar-servidor.command
    • Ou no Terminal: ./iniciar-servidor.sh

  Windows:
    • Dê duplo clique em: iniciar-servidor.bat

O navegador abrirá em: http://localhost:8080/

Arquivos de entrada:
  • index.html
  • Benini-Squad.html  (cópia do index, mesmo conteúdo)

───────────────────────────────────────────────────────────────
TRANSFERIR PARA OUTRO DESKTOP
───────────────────────────────────────────────────────────────

1. Copie a pasta inteira "Benini-Squad-Portable" (USB, nuvem, etc.)
2. Não copie só o .html — os assets estão nas subpastas:
   css/  js/  images/  videos/  models/  assets/
3. No outro computador, execute o script de servidor acima

───────────────────────────────────────────────────────────────
REQUISITOS
───────────────────────────────────────────────────────────────

• Navegador moderno (Chrome, Edge, Firefox ou Safari)
• Conexão com internet na primeira visita (CDNs: GSAP, Lenis,
  Three.js, Google Fonts DM Sans)
• Python 3 instalado (para o servidor local nos scripts inclusos)
  — alternativa: Node.js com "npx live-server" na pasta
